* STMicroelectronics Video Input Port driver

Required properties:
- compatible: Should be "st,sta_vip"
- input: Analog source input
- reg: Address and length of the register set for both VIP and SGA devices
- reg-names: Register name for VIP("vip_regs") and SGA("sga_regs")
  respectively
- interrupts: Should contain 3 GIC Interrupt lines, VIP(Line&Frame Interrupts)
  and SGA interrupt line

Optional properties:
- clocks: If present, the first clock should be the VIP main clock and the
  second clock should be SGA clock.
- clock-names: One name for each entry in the clocks property, the first one
  should be "apb_pclk" and the second one should be "sga_clk".
- pwr-gpio: If any, defines Power GPIO pins that power up the device
- reset-gpio: If any, defines Reset GPIO pins that resets the device

Examples:

	vip: vip@48800000 {
		compatible = "st,sta_vip";
		input = <0x2>;
		reg = <0x48800000 0x20000>,  /* VIP Registers */
		      <0x48700000 0x800>;
		reg-names = "vip_regs", "sga_regs";
		interrupts = <GIC_SPI 48 IRQ_TYPE_LEVEL_HIGH>,
			     <GIC_SPI 49 IRQ_TYPE_LEVEL_HIGH>,
			     <GIC_SPI 47 IRQ_TYPE_LEVEL_HIGH>;
		clocks = <&hclkvip>, <&hclkgae>;
		clock-names = "apb_pclk", "sga_clk";
		status = "disabled";
	};
